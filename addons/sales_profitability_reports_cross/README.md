# Sales and Profitability Reports

## Overview

The **Sales and Profitability Report** module allows you to generate sales and profitability reports filtered by user and a specific date range. It is ideal for monitoring user sales performance over a specific period.

## Features

- Generation of Sales Reports by User.
- Generation of Profitability Reports by User.
- Filtering by date range.
- PDF generation.

## Installation

To install this module, simply add it to your Odoo plugins directory and install it through the Odoo interface.

## Use

- After installing the module, go to the sales reports section. There you can filter sales by user and date range to generate the report.
