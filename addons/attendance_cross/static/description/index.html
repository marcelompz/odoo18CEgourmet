Biometric Attendance Integration

Automate and centralize employee attendance with Biometric biometric devices

Overview

This module allows you to synchronize employees and attendance records between Odoo and Biometric biometric devices. It enables both single and bulk employee registration, real-time attendance download, and detailed attendance reporting.

📋 Employee Registration

Register employees in Odoo with their biometric ID, assigned device, and formal photo for facial recognition devices.

⬆️ Bulk Upload

Upload all employees to Biometric devices in one click or upload individuals as needed.

⬇️ Attendance Download

Download attendance records from devices with customizable date ranges.

📊 Excel Reports

Generate detailed attendance reports in Excel format for payroll and compliance.

Key Features

  • Support for multiple Biometric biometric devices
  • Individual and bulk employee registration
  • Real-time attendance synchronization
  • Automatic processing to Odoo attendance records
  • Customizable Excel report generation
  • Device connection testing
  • Employee-device assignment management

Installation

  1. Install the module from Odoo Apps
  2. Go to Attendance → Biometric Devices
  3. Add your Biometric device configuration (IP, port, credentials)
  4. Test the connection using the "Test Connection" button
  5. Assign devices to employees in the HR module
  6. Upload employees to devices using bulk or individual upload
  7. Download attendance records as needed

Requirements

  • Odoo Community Edition 18.0
  • Python libraries: openpyxl (for Excel reports)
  • Biometric biometric devices with ISAPI support
  • Network connectivity between Odoo server and Biometric devices
Note: The actual Biometric API integration requires implementation of the ISAPI protocol. The current module provides the framework and interface for integration.
