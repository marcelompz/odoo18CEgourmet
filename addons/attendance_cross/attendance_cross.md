# Biometric Attendance Integration for Odoo

`attendance_cross`

## Automate and centralize employee attendance with Biometric biometric devices

This module allows you to synchronize employees and attendance records between Odoo and Biometric biometric devices. It enables both single and bulk employee registration, real-time attendance download, and detailed attendance reporting.

![Main interface of the Biometric integration module](media/image5.png)

## Employee Registration in Odoo

To begin, register an employee in the Odoo system with their personal details, including: full name, department, gender, biometric ID, assigned device, and a formal photo if the device supports facial recognition.

![Employee registration form in Odoo](media/image6.png)

## Register Employees to Device

Employees can be uploaded to Biometric devices using two methods:

- **Individual Upload:** From the employee form, click "Upload user to device".
- **Bulk Upload:** Go to Attendance → Biometric Device → Biometric Device and click "Upload all users to the device".

Always ensure the device connection is working using the "Connection Test" button before uploading.

![Device selection screen for bulk upload](media/image4.png)

## Download and Review Attendance

You can download attendance records from the device with a specified date and time range. This is done from the Biometric device form using the "Download Attendances" button.

![Download attendance records interface - Step 1](media/image3.png)

![Download attendance records interface - Step 2](media/image1.png)

## Generate Attendance Reports

After downloading attendance data, generate reports in Excel format from the Assistance menu. Select the records, click "Actions" → "Download attendance report", and choose the time period and device.

![Attendance report generation options](media/image2.png)
